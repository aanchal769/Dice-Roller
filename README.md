# 🎲 Dice Roller

A Dice Roller game built using Python and Pygame.

## Features
- 🎲 Roll two dice
- 🔊 Dice rolling sound
- 📜 Roll history
- 🔢 Roll counter
- 📊 Highest, Lowest and Average score
- 🔄 Reset button
- 🖱️ Button hover effect

## Technologies Used
- Python
- Pygame

## How to Run

### Install pygame

```bash
pip install pygame
```

### Run the game

```bash
python dice.py
```
